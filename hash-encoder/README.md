# Simple Hash Encoder

Python project to encode a file, linking to tampering detection and irreversibility.

## What it does:
* Takes user input of a path to a file.
* Ouputs the hash encoding of that file
  * Using SHA256.
 
## What I learnt:
* How SHA256 and hashlib are relevant to encoding files.
* The relevance of encoding in cybersecurity and tampering detection.
* Importance of breaking down larger files when encoding.
